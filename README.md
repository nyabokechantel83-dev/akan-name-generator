# Akan Name Generator

**Author:** Chantel 

## Description

A simple web app that turns your birth date into a unique Akan name based on the day you were born.

## Setup

### Option 1: Clone the repository

1. Clone the repository:

   ```
   git clone https://github.com/nyabokechantel83-dev/akan-name-generator.git
   ```
2. Navigate into the project folder

### Option 2: Create the project manually

1. Create a project folder
   Name it `akan-name-generator`

2. Inside this folder, create the following files:

   * `index.html` → Your webpage structure
   * `style.css` → For styling the webpage
   * `script.js` → Contains the logic for name generation

### Run the project

* Open `index.html` in your browser
